## Descripción

Este repositorio contiene la implementación de tres ejercicios relacionados con lógica difusa utilizando Python.

Los ejercicios corresponden al apartado:


## Ejercicio 1: Incertidumbre y Probabilidad

Archivo:

01_incertidumbre_probabilidad.py

Descripción:

Se realiza una simulación de 10 lanzamientos de un dado utilizando números aleatorios.

El programa calcula cuántas veces aparece un número mayor a 4 y obtiene una probabilidad estimada.



## Ejercicio 2: Funciones de Pertenencia

Archivo:

02_funciones_pertenencia.py

Descripción:

Se implementa una función de pertenencia triangular.

Se utiliza como ejemplo una temperatura cálida con los valores:

a = 15

b = 25

c = 35

El programa calcula el grado de pertenencia de una temperatura determinada.



## Ejercicio 3: Defuzificación por Centroide

Archivo:

03_defuzificacion_centroide.py

Descripción:

Se aplica el método del centroide para transformar valores difusos en un valor concreto.

Valores utilizados:

10, 20 y 30

Grados de pertenencia:

0.2, 0.5 y 0.8

Resultado esperado:

24.00
